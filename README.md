# Controle Financeiro

Projeto desenvolvido durante o curso de Java da DIO.

## Objetivo

Aprender os fundamentos da linguagem Java através da resolução de exercícios práticos e estruturados.

---

## Exercícios Desenvolvidos

### Exercício 1 - Nome e Idade
* **Recebe:** Nome e ano de nascimento.
* **Calcula:** Idade atual com base no ano vigente.
* **Exibe:** "Olá [Nome], você tem [X] anos."

### Exercício 2 - Verificação de Números
* **Recebe:** Um número base de referência inicial.
* **Valida:** Utiliza um laço de repetição (`while`) para solicitar novos números continuamente, garantindo que o valor informado pelo usuário atenda à condição de validação estabelecida.
